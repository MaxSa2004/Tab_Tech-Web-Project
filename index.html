How to play T창b

Introduction

T창b is a two player board game, with 4 rows and a variable odd number of columns that you can choose at the configuration panel.

Player Perspective

In a board with n columns, each player has n blue/red board pieces, depending on the player.

Dice

The game dice used is a combination of 4 wooden sticks, each one with two sides, one darker and one lighter.

To throw it, we grab the sticks all together and let them go, making them fall on one of its side.

The value is known by the number of lighter sides that are facing up, and each value corresponds to the deslocation of a piece.

If the value is 1,4 or 6, the player can throw the dice again. Warning: there is no value 5!!

Moving the pieces

After throwing the dice, you select the piece you want to move, and move it n spots, being n the value of the dice.
